Introdução

Introdução

  • O Macintosh Operating System (Mac OS) é a de denominação de um sistema operacional padrão dos computadores Macintosh, é um sistema operativo proprietário baseado no kernel Unix titulado XNU.

MAC OS

MAC OS

  • Mac OS foi desenvolvido pela Apple, e seu surgimento ocorreu em 1984, junto ao lançamento do primeiro Macintosh.
  • Ficou conhecido por ser o sistema operacional a popularizar o uso da interface gráfica.
  • Hoje ele é o Mac OS X cuja introdução aconteceu em 2001.

MAC OS

  • Tem uma base UNIX e foi desenvolvido com uma série de diferenciais justamente para os usuários aproveitarem os aparelhos da linha Mac.
  • Tem um aspecto moderno, com multijanelas eficiente, efeitos de transição em ações simples, como a de excluir algum conteúdo ou passar o mouse por cima dos ícones da barra inferior.

MAC OS X

  • Ficou conhecido por ser o sistema operacional a popularizar o uso da interface gráfica.

Versões

História

Cheetah

  • O primeiro sistema da linha Mac OS X foi anunciado em janeiro de 1999, com o nome Mac OS X Server 1.0.
  • Era vendida a US$ 129.
  • O sistema sofreu muitas críticas no tempo de resposta da interface, estabilidade do sistema e problemas de compatibilidade com hardware.

Puma

  • Em Setembro de 2001, a Apple lançou o Mac OS X 10.1, codinome “Puma”.
  • A versão foi fornecida gratuitamente como atualização do Cheetah.
  • Como mudanças, foi melhorado o desempenho geral do sistema e foi adicionado um suporte mais eficiente à gravação de CDs e DVDs.

Jaguar

  • Em 23 de Agosto de 2002, a Apple lançou mais uma versão, o Mac OS X 10.2, codinome “Jaguar”.
  • Teve melhorias de desempenho e estabilidade.

Panther

  • Em Outubro de 2003, a Apple lançou o Mac OS X 10.3, codinome “Panther”.
  • O sistema facilitou a visualização de janelas do usuário.
  • Lançamento da ferramenta de programação Xcode.

Leopard

  • Mac OS X 10.5 “Leopard” foi lançado em outubro de 2007.
  • O Leopard introduziu um design novo da área de trabalho.
  • O desempenho foi otimizado e foi introduzido a Time Machine, ferramenta de backup da Apple.

Snow Leopard

  • O Mac OS X 10.6, codinome “Snow Leopard” foi lançado em junho de 2009.
  • O objetivo principal foi melhorar o desempenho e reduzir a utilização de memória.

Mountain Lion

  • Em Fevereiro de 2012, a Apple anunciou o OS X 10.8, codinome “Mountain Lion”.
  • Diferentemente das versões anteriores, a Apple preferiu chamar o sistema simplesmente de OS X, em vez de Mac OS X.

Yosemite

  • Em outubro de 2014, a Apple anunciou o “Yosemite”.
  • Melhorias foram o novo Safari que está até seis vezes mais rápido que o Google Chrome.
